# Python Games 🎮

A collection of beginner Python games built while learning Python.

## Games Included

### 🪨 Rock Paper Scissors

Play the classic Rock, Paper, Scissors game against the computer.

### 🔢 Guess the Number

Try to guess the randomly generated number in as few attempts as possible.

### 🪙 Coin Flip

Simulates flipping a coin and randomly returns Heads or Tails.

### 🎲 Dice Roll

Simulates rolling a six-sided dice using Python's random module.

## Skills Practiced

* Variables and data types
* User input
* Conditional statements (`if`, `elif`, `else`)
* Loops
* Functions
* Dictionaries
* Random module
* Basic game logic

## How to Run

1. Clone the repository:

```bash
git clone <repository-link>
```

2. Open the project folder.

3. Run any game:

```bash
python game.py
```
